# 状態遷移図
## タイマー
```
                ┌───────────────┐
                │               │
                │    再開       │
                │               │
                ▼               │
[未開始] ──開始──▶ [計測中] ──停止──▶ [停止中]
   │                │   ▲            │
   │                │   │            │
   │                │   │            │
   │                └─自動終了───────┘
   │
   └────────────終了────────────────▶ [終了]

```
## 例外ルール
### running中に「開始」
| 判定      | 動作              |
| ------- | --------------- |
| running | エラー             |
| 理由      | Sessionは同時に1つまで |
### paused中に「停止」
| 判定     | 動作        |
| ------ | --------- |
| paused | 無視 or エラー |
| 理由     | 既に停止中     |
### end中に「再開」
| 判定    | 動作             |
| ----- | -------------- |
| ended | 新Session作成を促す  |
| 理由    | Sessionは再利用しない |
### idle中に「停止」
| 判定   | 動作       |
| ---- | -------- |
| idle | 無視       |
| 理由   | 何も起きていない |
